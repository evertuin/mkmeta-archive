# mkmeta-archive

本仓库用于归档和发布多个独立的 HTML 页面。这些页面以静态 HTML 文件的形式存放，不针对任何特定域名或网站。仓库旨在提供一种简洁、可维护的方式来保存和展示各类独立页面内容。

## 目录说明

```
mkmeta-archive/
├── pages/          # 存放所有归档的 HTML 页面文件
└── README.md       # 本文件
```

- **pages/**：该目录下包含所有已归档的 HTML 页面。每个文件均为独立的、自包含的 HTML 文档，可直接通过浏览器打开查看。

## 页面归档说明

- 所有页面均为静态 HTML，不依赖外部资源（除非页面内自行引用）。
- 文件命名遵循简洁、无歧义的原则，通常使用英文或拼音。
- 每个页面代表一个独立的内容单元，彼此之间无直接关联。
- 归档内容可能随时间更新或补充，但不会对已有页面进行修改（除非必要修正）。

## 维护说明

- 本仓库由维护者不定期更新，新增页面或修正已有内容。
- 如发现页面显示异常或包含不适当内容，欢迎通过 Issue 反馈。
- 不接受未经协调的大规模批量提交或与归档主题无关的页面。
- 仓库内容仅供存档与展示，不构成任何形式的承诺或保证。

## 使用方式

您可以直接浏览 `pages/` 目录下的 HTML 文件，或将其克隆到本地后通过浏览器查看。

```bash
git clone https://github.com/your-username/mkmeta-archive.git
cd mkmeta-archive/pages
```

然后双击任意 `.html` 文件即可在浏览器中打开。
